# 📊 Power BI Business Analytics Projects

> **A collection of interactive Power BI dashboards analyzing Call Center Performance, Customer Churn, and Diversity & Inclusion to generate actionable business insights.**

---

# 📖 Project Overview

This repository showcases three business analytics case studies developed using **Power BI**. Each dashboard focuses on a different business domain and demonstrates how data visualization can support operational improvements and strategic decision-making.

The projects analyze customer service performance, customer retention, and workforce diversity using interactive dashboards and business-focused KPIs.

---

# 🎯 Projects Included

## 📞 1. Call Center Performance Analysis

### Objective

Analyze call center operations to monitor agent performance, customer satisfaction, call handling efficiency, and overall service quality.

### Key Insights

* Call volume trends
* Customer satisfaction analysis
* Average call handling performance
* Agent performance comparison
* Call resolution metrics

---

## 📉 2. Customer Churn Analysis

### Objective

Identify customer churn patterns and understand the factors influencing customer retention.

### Key Insights

* Customer churn overview
* Churn distribution
* Customer demographics
* Service usage analysis
* Customer retention trends

---

## 🌍 3. Diversity & Inclusion Analysis

### Objective

Evaluate workforce diversity metrics and support inclusive workplace decision-making.

### Key Insights

* Gender diversity analysis
* Hiring trends
* Promotion analysis
* Employee distribution
* Diversity KPIs
